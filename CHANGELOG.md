## [0.1.0] - 2024-01-15

### Added
- Initial release
- `Clack.text` - Text input prompt
- `Clack.password` - Masked password input
- `Clack.confirm` - Yes/No confirmation
- `Clack.select` - Single choice selection
- `Clack.multiselect` - Multiple choice selection
- `Clack.spinner` - Animated loading indicator
- `Clack.log.*` - Styled log messages (info, success, warn, error, step)
- `Clack.note` - Boxed note display
- `Clack.intro` / `Clack.outro` - Session start/end markers
- `Clack.cancel?` - Check for user cancellation
- `clack-demo` executable for showcasing all features
- Zero runtime dependencies (stdlib only)
